# Acme.Leads

OctoberCMS v3+ plugin: contact leads form with backend management.

> **Status:** work in progress — full README will be added in the final phase.

## Install

1. Clone this repo into `plugins/acme/leads` of your OctoberCMS project:
```bash
   git clone <repo-url> plugins/acme/leads
```
2. Run migrations:
```bash
   php artisan october:migrate
```
